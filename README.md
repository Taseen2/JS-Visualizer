# JS Visualizer 🌊

JS Visualizer is a modern, interactive educational platform designed to teach complex JavaScript array methods through beautiful, step-by-step visualizations. Built with a focus on UX/UI polish, it transforms abstract coding concepts into a tangible visual experience.

## ✨ Features

### 1. Reduce() Tally Visualizer
Watch in real-time as the `reduce()` method iterates through an array to build a frequency map.
- **Live Accumulator State:** See the object update dynamically at each step.
- **Speed Control:** Adjust animation speed from lightning-fast to slow-motion for deep learning.
- **Visual Highlighting:** The current element "glows" as it's being processed.

### 2. Smart Sorting Visualizer
Learn how to sort strings while intelligently ignoring articles like "The", "A", and "An" using Regular Expressions.
- **Step-by-Step Transformation:** View the original list, the "stripped" version (Regex-applied), and the final sorted result.
- **Regex Highlighting:** Articles are visually struck through to show exactly what the algorithm ignores.

### 3. Interactive Playgrounds
Mini code editors where you can run your own snippets.
- **Live Output:** Immediate feedback as you type.
- **Error Handling:** Helpful error messages for malformed arrays.
- **Copy Code:** Quick access to copy your playground snippets.

### 4. Premium UI/UX
- **Glassmorphism Design:** Modern aesthetic with backdrop blurs, neon gradients, and floating background blobs.
- **Dark Mode First:** Beautiful dark theme by default with a smooth light mode toggle.
- **Micro-interactions:** Ripple effects, toast notifications, and scroll-reveal animations.
- **Keyboard Shortcuts:**
  - `Space` / `Enter`: Start visualizer
  - `Esc`: Reset visualizer
  - `Alt + T`: Toggle Theme
--- 

- **HTML5:** Semantic structure.
- **CSS3:** Custom properties (variables), Flexbox layout, and advanced animations.
- **Vanilla JavaScript:** ES6+ modular logic (no frameworks or libraries).
